Introduction

JavaScript is a versatile programming language primarily used for web development. It allows developers to create interactive and dynamic web pages.

Example:

<script>
alert('Hello, World!');
</script>

Basics

Variables

Use let, const, or var to declare variables.

let name = "John";
const age = 30;
var isDeveloper = true;

Functions

Functions allow you to encapsulate reusable logic.

function greet(name) {
    return `Hello, ${name}`;
}
console.log(greet("Alice"));

Advanced Concepts

Promises

Promises are used for asynchronous programming.

const fetchData = () => {
    return new Promise((resolve, reject) => {
        setTimeout(() => resolve("Data fetched!"), 2000);
    });
};
fetchData().then(console.log);

Async/Await

Simplified syntax for working with promises:

async function getData() {
    const data = await fetchData();
    console.log(data);
}
getData();

Events

JavaScript events allow you to execute code when something happens on the web page.

document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

DOM Manipulation

The Document Object Model (DOM) represents the structure of a web page. JavaScript can manipulate the DOM to change content, styles, and structure dynamically.

document.getElementById("myDiv").innerText = "Hello, DOM!";

ES6 Features

ES6 introduced several new features, such as template literals, arrow functions, and classes.

const add = (a, b) => a + b;
console.log(add(2, 3));

Debugging

Debugging is the process of identifying and fixing errors in your code. Use tools like console.log and browser developer tools.

console.log("Debugging in progress...");
